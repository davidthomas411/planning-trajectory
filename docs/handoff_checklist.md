# Handoff Checklist

Use this checklist when handing the project to a student.

- [ ] Repo runs locally with `python3 src/dashboard/app.py`.
- [ ] `.env.template` exists and is up to date.
- [ ] `.gitignore` blocks `data/derived/` and CSV exports.
- [ ] `scripts/run_pipeline.sh` executes without errors.
- [ ] `scripts/export_csv.py` produces `data/derived/exports/*.csv`.
- [ ] `docs/sample_data/` contains non-PHI examples.
- [ ] `draft_abstract.md` reflects the latest results.
